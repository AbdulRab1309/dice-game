# 🎲 Dicee Game

A fun, interactive two-player dice rolling game built with vanilla HTML, CSS, and JavaScript.

## 🎮 How to Play

- **Click anywhere** on the page to roll both dice
- **Press Enter** on your keyboard to roll both dice
- The player with the higher dice number wins!
- If both players roll the same number, it's a draw 😵‍💫

## 🛠️ Built With

- **HTML5** — Page structure
- **CSS3** — Styling with Google Fonts (Lobster & Indie Flower)
- **JavaScript** — Game logic and DOM manipulation

## ✨ Features

- 🎲 Random dice roll for two players
- 🖱️ Click-to-roll interaction
- ⌨️ Keyboard support (Enter key)
- 🏆 Automatic winner detection
- 🎨 Clean, responsive UI with custom fonts
